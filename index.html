GitHub Workflow

What is a commit?

A commit is used to save changes to a local repository.

What is a branch?

A Git branch is an independant line of development. Branches can be merged into one branch, as long as they belong to the same repository.

What is a pull request?

A pull request is used to download and fetch content from a remote repository, immediately updating the local repository to match the content.

Unix Shell Workflow

How to list files in a directory

When navigating through folders. Entering the "ls" command will list files and directories.

How to create a new directory

To create a new directory, navigate to your desired directory that you would like to be the parent of the new one. After doing this, use the "mkdir" command followed by the name. For example: "$ mkdir new-directory".

How to create and remove files

To create a file in a directory, the "touch" command is used. For example: "$ touch index.html".
